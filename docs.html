Documentación de la API de Inventario

Esta API permite gestionar los artículos de un inventario de forma sencilla.

Modelo de Datos: Articulo

{
  "codigo": "integer (único)",
  "nombre": "string",
  "marca": "string",
  "unidades": "integer",
  "nombre_bodega": "string"
}

Endpoints

GET /articulos/

Obtiene la lista de todos los artículos registrados.

POST /articulos/

Crea un nuevo artículo en el inventario.

Cuerpo de la petición: El objeto Articulo (sin ID).

GET /articulos/{codigo}

Obtiene los detalles de un artículo específico mediante su código.

PUT /articulos/{codigo}

Modifica un artículo existente identificado por su código.

DELETE /articulos/{codigo}

Elimina un artículo del inventario.

Documentación Interactiva (Swagger)

Una vez la API esté en ejecución, puedes acceder a la documentación interactiva en:

  • Swagger UI: http://localhost:8000/docs
  • ReDoc: http://localhost:8000/redoc
Desarrollado con FastAPI y SQLModel
